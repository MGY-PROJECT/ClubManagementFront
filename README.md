##### 로직은 in-file이라는 class를 만들어서 display : none; 으로 준 뒤에, 배경 이미지가 선택이 되면 in-file 클래스에 .event가 추가되면서 display : block으로 보여지게 하고, in-file에 파일 경로를 background-image나 img에 적용 시켜서 배경화면이 나오게 하도록 해볼예정 안되면 경호가 해주겠지 ㅎㅎㅎ
